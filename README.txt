Proyecto Tdp
En este proyecto se encuantran implementadas las siguientes funcionalidades:
- Lista de valores: se muestra como pantalla principal una lista con los valores almacenados en la aplicacion. Cada valor puede ser eliminado o editado si fuese necesario.
- Cargar valor nuevo: permite la carga de un valor nuevo a la lista, indicando valor, fecha, hora, momento de la medicion, y una observacion si es necesario.
- Estadisticas: Se elige de una lista de parametros la cantidad de dias que se desea visualizar y el momento del dia, y se muestra: el valor promedio en ese periodo, la cantidad de mediciones, la cantidad de mediciones promedio, y la cantidad de hipo e hiperglucemias (valores fuera del rango [70,170]).
- AcercaDe: Se muestra una pequeña descripcion sobre la aplicacion. 

Se encuentra implementado la persistencia de datos de los valores.
